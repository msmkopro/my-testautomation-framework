This project is a test automation framework developed rom scratch using Java and Selenium
It is designed to provide a scalable, maintainable and reusable structure for automating web applications

PURPOSE

The goal of this framework is to demonstrate understanding of
Test automation design principles
Framework architecture
Real-world QA automation practices

TECH STACK

Java
Selenium WebDriver
TestNG
Maven
Log4j

FRAMEWORK DESIGN

The framework follows best practices such as

Page Object Model (POM)
Separation of test logic and page logic
Reusable utility methods
Centralized configuration management

FEATURES

Structured test execution using TestNG
Logging with Log4j
Easy test scalability
Clean and maintainable codebase

FUTURE ENHANCEMENTS

Parallel test execution
CI/CD integration (GitHub Actions)
Screenshot capture on test failure
Cross-browser testing
